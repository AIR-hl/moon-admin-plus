# my-springsecurity-plus
#### 介绍
这是一款基于SpringBoot+SpringSecurity的RBAC权限管理系统

#### 技术选型
1、SpringBoot  
2、MyBatis    
3、SpringSecurity  
4、MySql  
5、Druid   
6、Swagger    
8、Redis  
9、JWT  
10、[Pear Admin Layui](https://gitee.com/Jmysy/Pear-Admin-Layui)  
等
#### 快速使用

- 下载maven项目
 
- 导入idea
 
- 导入docs文件夹下sql文件到数据库
 
- 修改数据库配置文件的路径，用户名等信息
 
- 运行
